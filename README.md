# baseguard
100% client-side token approval revoker for Base Network. No backend, no data tracking.
##  Why BaseGuard?
Most Web3 security tools require you to connect your wallet to their backend servers, logging your IP, address, and transaction history. BaseGuard is different: **your data never leaves your browser.**

##  Features
- **Zero Backend:** Runs entirely in your browser using Vanilla JS and Ethers.js v6.
- **Privacy-First:** No analytics, no cookies, no data retention.
- **Micro-transaction Model:** Basic scan is free. Batch revocation is a one-time 1 USDC payment directly to the developer's wallet via smart contract (no middlemen).
